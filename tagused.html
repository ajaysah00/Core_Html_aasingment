• <Html>
  • Defines the root element of an HTML document
  • Contains all other HTML elements.
  • <link>
  • Specifies external resources like stylesheets (CSS) to be used with the HTML document.
  • <style>
  • Contains inline CSS styles or links to internal CSS stylesheets.
  • <body>
  • Represents the content of the HTML document that is visible in the browser.
  • <div>
  • A generic container for grouping and styling HTML elements.
  • <a>
  • Creates hyperlinks to other web pages or resources.
  • <ul>
  • Defines an unordered (bulleted) list.
